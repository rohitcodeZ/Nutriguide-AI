# Nutriguide-AI
AI-driven nutrition assistant built with MERN Stack that helps users track calories, analyze food, generate healthy recipes, and receive personalized diet plans based on goals, weather, and location.

# NutriGuide AI

AI-powered nutrition and diet planning platform built using the MERN Stack.

## Features
- Personalized Diet Plans
- AI Recipe Generation
- Food Analysis
- Calorie Tracking
- Location-Based Suggestions
- Weather-Based Recommendations

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js
- JWT Authentication
- DeepSeek/Gemini AI
